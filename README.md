# prevent-window-scroll

a light script with prevent body scroll when part scroll, and it has no dependancy.

弹层局部滚动时，阻止页面整体滚动

特点：
> * 1、无依赖，原生js实现；
> * 2、经测试兼容90%以上的浏览器；
> * 3、经测试，小米max2自带浏览器和UC下快速滑动会动一点点，不过已经基本上满足方案要求；
